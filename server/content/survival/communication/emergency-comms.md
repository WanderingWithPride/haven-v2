## Signal Methods

### Visual Signals
- **3 of anything** = universal distress (3 fires, 3 whistle blasts, 3 gunshots)
- **SOS** in ground: use rocks, branches, or trampled snow. Make letters 10+ feet tall
- **Mirror signal:** Flash sunlight toward aircraft/rescuers. Even a phone screen works
- **Bright clothing/markers:** Hang on trees, place on open ground
- **Smoke:** Green vegetation on fire = white smoke (daytime). Dry wood = black smoke (against snow)

### Audio Signals
- **3 short whistle blasts** = distress
- **6 whistle blasts** (1 per minute) = Alpine distress signal
- Whistle carries **much further** than shouting (up to 1.6 km vs 200m)

## Improvised Communication

### Ground-to-Air Signals (ICAO Standard)
| Symbol | Meaning |
|--------|---------|
| V | Need assistance |
| X | Need medical help |
| N | No / Negative |
| Y | Yes / Affirmative |
| → | Heading this direction |
| I | Serious injury |
| F | Need food and water |
| LL | All is well |

Make symbols **minimum 10 feet** tall, visible from above.

### Dead Drops / Message System
If you need to leave messages for others:
- Predetermined locations (tree hollows, rock piles)
- Waterproof container (plastic bag)
- Include: date, heading, condition, next planned location
- Use code if needed for security

## Phone as Communication Tool (When Network Down)
- **WiFi Direct:** Most phones can create point-to-point connections
- **Bluetooth:** Short range, ~30m but no infrastructure needed
- **LAN Server:** Your CyberDeck can host a chat room for nearby devices on the same WiFi
- **Hotspot mode:** Create WiFi network even without internet
- **Record messages:** Audio recordings to hand off physically (USB/SD card)
