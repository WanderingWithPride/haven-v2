# Security & Threat Assessment

## Situational Awareness (Color Code System)

| Level | State | Description |
|-------|-------|-------------|
| ⬜ White | Unaware | Relaxed, not paying attention: AVOID this state |
| 🟢 Green | Aware | Relaxed but alert, scanning environment |
| 🟡 Yellow | Alert | Specific threat identified, preparing response |
| 🟠 Orange | Alarm | Threat imminent, ready to act |
| 🔴 Red | Action | Engaged in response (fight, flight, freeze) |

**Goal: Stay in Green state at all times.**

## Urban Threat Assessment

### Before Moving
1. **Route planning:** Primary and alternate routes
2. **Time:** Safest travel times (varies by situation)
3. **Communication:** Inform someone of your plan
4. **Supplies:** Water, food, first aid, documents
5. **Observation:** Watch for 15 min before moving through new areas

### Danger Signs
- Unusual silence (people hiding = something they know you don't)
- Barricades, checkpoints manned by unknowns
- Smoke, gunfire, explosions (even distant)
- Abandoned vehicles blocking roads
- Groups moving with purpose/weapons
- Infrastructure damage (bridges, power lines)

## Evacuation Planning

### Bug Out Bag Essentials
| Category | Items |
|----------|-------|
| Water | Bottles + purification method |
| Food | 3 days minimum, calorie-dense |
| Shelter | Tarp, space blanket, paracord |
| Fire | Lighter, matches, ferro rod |
| Medical | First aid kit, medications |
| Nav | Compass, physical map, this phone |
| Tools | Knife, multitool, duct tape |
| Comms | Whistle, mirror, radio |
| Docs | ID copies, cash (small bills), contacts list |
| Power | Solar charger, battery pack |

### Route Planning
1. **Primary:** Fastest route to safe location
2. **Alternate:** Different direction, avoids likely obstacles
3. **Contingency:** If first two fail
4. **Emergency:** Final option, may involve wilderness travel

Mark on your offline maps:
- ✅ Water sources
- ✅ Shelter options
- ✅ Resupply points
- ✅ Danger zones
- ✅ Rally points

## Personal Security

### Gray Man Concept
Blend in, don't attract attention:
- Wear common, neutral clothing
- No visible expensive items (watches, phones)
- Match local behavior and pace
- Avoid eye contact with aggressive individuals
- Don't display supplies/weapons openly

### Document Backup
Store digitally (encrypted on this device):
- ID/Passport copies
- Medical records
- Insurance documents
- Property documents
- Emergency contacts
- Bank details
- Family photos for identification

### Communication Security
- Use code words with family for status updates
- Pre-agreed meeting points if separated
- Radio check-in schedule
- Don't broadcast your location/situation openly

## Conflict Zones: Do's and Don'ts

### DO
- Stay low, stay quiet
- Move during low-activity hours
- Keep to walls and cover
- Travel in small groups (2-4 people)
- Know the difference between cover (stops bullets: concrete, engine block) and concealment (hides you: bushes, curtains)

### DON'T
- Don't approach checkpoints unpredictably
- Don't run (triggers pursuit response)
- Don't photograph military/security forces
- Don't carry anything that could be mistaken for a weapon
- Don't resist if detained: stay calm, be cooperative
